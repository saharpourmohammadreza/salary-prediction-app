# 💰 Salary Prediction App (2022 Stack Overflow Dataset)

This project is a machine learning web application that predicts software developer salaries based on demographic and professional information using a trained **Linear Regression model**.

The app is deployed using **Streamlit** and hosted on **Hugging Face Spaces**.

---

## 🚀 Live Demo

👉 Try the app here:
https://huggingface.co/spaces/SaharPMR/Salay_Prediction

---

## 📊 Project Overview

The model is trained on the **Stack Overflow Developer Survey 2022 dataset** and estimates annual salary based on:

* Country
* Education Level
* Years of Professional Experience
* Employment Type

---

## 🧠 Machine Learning Model

* Algorithm: Linear Regression
* Library: Scikit-learn
* Input features:

  * Country
  * Education Level
  * Years of Experience
  * Employment Type

## ⚙️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/saharpourmohammadreza/salary-prediction-app.git
cd salary-prediction-app
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run SalaryApp.py
```
